# FSP Finals Frontend

## How to dev?

```Bash
git clone git@github.com:csit-sgu/fsp-finals-frontend.git
cd fsp-finals-frontend
npm i
npm run prepare
npm run dev
```

## How to format?

```Bash
npm run lint:fix
```
